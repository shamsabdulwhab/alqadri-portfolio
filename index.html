Welcome to my portfolio


This portfolio is all about showing my top projects. To organize my work, I'm using the Double Diamond method. It's split my work into 4 phases (Discover- Define - Develop - Deliver).

  • Website for a Greek artist: Me and my team had the task to create a website that would showcase the art of Erion, a Greek artist. We focused our project on the client's wants and needs as well as possible users of the website. We interviewed stakeholders, performed A/B testing, and created an empathy map and user personas. After that, we created a Figma design and prototype, finalizing a working website (that we coded usingHTML, CSS, and Javascript).
  • Turorial Website: At the start of our project, we did a lot of research.Recognizing students and teachers as our stakeholders, we conducted interviews and surveys to know their needs and preferences. Through this process, we created mood boards, Point of View (POV) statements, and How Might We (HMW) questions, refining our understanding of the project scope. Then we entered the Define phase, where we synthesized our findings to develop a clear project direction. Subsequently, in the Develop phase, we translated our insights into action by creating a Figma design, ensuring alignment with user needs and project objectives. Utilizing HTML, CSS, and JavaScript alongside React, and connecting to a Content Management System (CMS) for the backend. The CMS makes it easy for our clients to change the website's content without the need to code.
  • Frontend Assignment : I created a fuly functioning blog using Node.js, Express & MongoDB. I've added user authentication, allowing users to log in using the provided credentials. The blog is connected to MongoDB, enabling efficient storage and retrieval of data. I also add a search bar if the user wants to search for an article. Users need to log in if they want to edit , add or delete articles.
